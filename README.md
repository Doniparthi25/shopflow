# shopflow
A microservices-based e-commerce platform.

## Architecture

API Gateway
User Service
Order Service
Payment Service
Kafka
PostgreSQL
Redis

## Setup

1. Clone repo
2. Run docker-compose
3. Start services

## Services

| Service | Port |
|--------|------|
| user-service | 8081 |
| order-service | 8082 |
| payment-service | 8083 |
